Knocking Bird

1. What is it?

This Java/Swing application has arbitrary oval area of images swinged. It is similar to both Wobble on iPhone and xx-cake! on MS-Windows. The multi-thread-oriented renderering routine allows you employ full of your CPU resource, and this application can also show frame-rates, which could use as a benchmark.

5. Installation / Uninstallation

Unarchive KnockingBird.zip, and double-click KnockingBird/KnockingBird.jar if you use either MS-Windows or Macintosh. You can also start it via CLI by typing the following command:

java -jar KnockingBird.jar
